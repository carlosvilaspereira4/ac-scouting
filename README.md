# Atlético Cabeceirense — Scouting App

## Deploy em 2 minutos (Vercel)

As credenciais Firebase já estão configuradas. Só precisas de:

### Passo 1 — GitHub

1. Vai a https://github.com/new e cria um repositório novo (pode ser privado)
2. Faz upload de todos os ficheiros desta pasta para o repositório

### Passo 2 — Vercel

1. Vai a https://vercel.com → liga com a tua conta GitHub
2. **Add New Project** → importa o repositório que acabaste de criar
3. Clica **Deploy** — em ~2 minutos tens o link público!

### O link fica tipo:
`https://ac-scouting.vercel.app`

Partilha com quem quiseres — todos veem os mesmos relatórios em tempo real.

---

## Desenvolvimento local (opcional)

```bash
npm install
npm run dev
```

Abre http://localhost:3000
